# Species ASCII Art

All 18 species with `{E}` as eye placeholder. Rendered by substituting the companion's eye symbol for each `{E}`.

## Rendering

To render a companion:

1. Select the species ASCII template
2. Replace all `{E}` with the companion's eye symbol
3. If the companion has a hat, prepend the hat ASCII above the head
4. Apply color if the companion has a non-default color

Example (blob with `·` eyes and crown hat):
```
  \^^^/
 .----.
( ·  · )
 '----'
```
